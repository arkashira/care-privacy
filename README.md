# 🚀 care-privacy
**Power developers with simple, intuitive consent management for AI services.** care-privacy is a basic web application that presents users with a consent form interface where they can enable or disable different data categories before submitting their consent.

## Why care-privacy?
* **Easy to use**: Simple web UI with toggle switches for various data types.
* **Customizable**: Allows users to toggle data categories before submission.
* **In-memory storage**: Stores records in a temporary in-memory database.
* **Basic testing**: Includes basic pytest tests for functionality.
* **Research-focused**: Ideal for research or development environments.
* **AI service integration**: Designed for technical developers or researchers working on AI systems.
* **Data privacy**: Focuses on data-privacy, making it a great solution for projects requiring basic consent management.

## Feature Overview
| Feature | Description |
